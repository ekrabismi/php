C) PHP Codes Explanation - top

In this section, I will explain in-depth on how to use some of the built-in functions and how you can re-use some of these functionalities to implement your own functions. All files are referred to the files inside the includes folder.

  1. ajax-handler.php

    This is a handler for all AJAX calls. This can be called as a POST or GET method. If you would like to add more functionalities, you must specify the ajax_object (explained below) parameter to be used. Furthermore, you can specify as many parameters as you like.

    • ajax_object - a must have parameter to know the type of operation to be called.

  2. commons.php

    This file contains all the common PHP codes and will be called at the very beginning. Example of such calls are: starting session, check if user is logged in, create database connection, etc. You can add more functionalities if you like.

  3. footer.php

    This is the footer of all the pages in the control panel.

  4. functions.php 
    db_connect()

    This creates a connection to database to a pre-configured settings. You generally do not have to change any code inside this function.

    get_current_page_name()

    When this function is being called, it will return the current file being accessed. If a user is currently accessing the page "http://www.messaging-system.com/message.php", this function will return "message.php." This function is used heavily in the top navigation menu for highlighting purposes.

    sanitize($input)

    This function will "sanitize" input(s) to prevent SQL injection.

    • input - can be variable (of any type) or an array.

    unsanitize($input)

    This function will "unsanitize" input(s) to be displayed to end-user.

    • input - can be variable (of any type) or an array.

    encrypt($str, $key = 'YOUR_KEY')

    This function will encrypt a given string and return the output.

    • str - string to be encrypted.
    • key - specify this to create a unique text. MUST match the key used in decrypt function.

    decrypt($str, $key = 'YOUR_KEY')

    This function will decrypt a given encrypted string and return the output.

    • str - string to be decrypted.
    • key - specify this to create the original text. MUST match the key used in encrypt function.

    user_exist($username)

    This function will check if a user exist in database or not. It will return true or false.

    • username - username to be checked.

    validate_user($username, $password)

    This function will validate the given username and password against database information. It will return true or false.

    • username - username to be validated.
    • password - password to be validated.

    message_exist($message_id, $from = '', $target_type = '', $target_username = '')

    This will check if a message exist or not. This can also further in the "message_sender" and "message_recipient" table if the specify the target parameters

    • message_id - the id to be checked
    • from - the table to be retrieved from
    • target_type (optional) - Specify either "message_sender" or "message_recipient".
    • target_username (optional) - The username to be checked.

    recipient_exist_address_book($username, $recipient)

    This will check if a recipient already exists in the sender's address book. It will return true or false.

    • username - the address of this user will be checked.
    • recipient - the username to be checked.

  5. header.php

  6. message-header.php

    This is the notification box used in the control panel.
